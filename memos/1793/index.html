大家好～ 当时建这个群好像就是想做公开笔记 Everentry（https://everentry.com/ ），绕了一圈，发现完美满足我需求的其实是一个简单灵活的博客系统，最近终于有空发布了： Jant（https://jant.me ）。



核心想法是降低公开写作的摩擦，帖子默认只在主页展示，精选之后才推送 RSS，甚至可以只是公开发布，连首页也不显示。界面用类似 Twitter 的体验，Note / Link / Quote 作为一等公民，内容自动同步到自己的 GitHub 仓库（一个完整的静态 Hugo 站点）。



我在博客里也写了一个比较完整的介绍：https://www.owenyoung.com/jant



开源地址：https://github.com/jant-me/jant

官网: https://jant.me

demo: https://demo.jant.me , 或者我的博客： https://www.owenyoung.com



欢迎有需求的朋友多多试用~ 界面大概如图
