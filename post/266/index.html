野草—写在七月的尾巴

By LeetaoGoooo on July 31, 2025

今天算是沾了台风的光又家居家办公了一天，居家办公效率有的时候是个谜，幸运地是，今天的效率还算蛮高的，截止到下班前把之前遗留的一些问题解决掉了。工作的问题是解决了，但是不过总觉得最近自己的状态有些许的不对劲。

晚上过后，又再次习惯性地坐在了电脑前，我在想接下来的几个小时应该做些什么。看着看板上一堆 TODO，想着再写会代码，反常的提不起兴趣来，那种奇怪的感觉又出现了，我隐约的察觉到了什么，不过转瞬即逝。

那就不写代码了，看会书吧，巧的是，昨天刚把《我胆小如鼠》读完，陷入了不知道读什么的困境。 我搜了一下微信读书七月份的书单，浏览了一下书单，没有一本书激起我的兴趣。无奈之下，我开始打卡自己的书架看看有没有什么现在想读的书，看了一下今年的书单，有几本竟然还没有加入书架，随手加入了书架。昨天刚读完的书也不在书单之中，便把它添加到书单里去了。加入书架的书有：《终身成长》、《人之天性》都是一些帮助人更好的成长的书，可惜今天的我对成长不感兴趣。

我漫无目的的开始在书架里浏览之前加入的书，突然翻到一本书名叫《在忙乱的世界找回平静》，那一瞬间我突然意识到了，我最近陷入了一种忙碌的困境，持续地忙碌在我无意识中形成了一种思维惯性，以至于我突然没有办法好好享受今晚的宁静。当忙碌成为生活的常态，在忙乱中找回平静，享受平静就会成为一种难能可贵的能力。

生活总会充斥了各种各样的烦恼和压力，忙碌似乎有种魔力可以让我们解决掉这些问题。一旦忙起来，脑子就会没有时间去思考这些问题，当别人问起来，就可以冠冕堂皇的说，没看见我很忙吗，没有时间处理这些问题。事实就是这样，问题是被拖延了，而不是消失了。

过分地沉迷在忙碌节奏里，会让人变得浮躁起来。缓过神来，我突然意识到我认真地写一点编程之外的文字了，没有自己的思考，放弃思考是一件很可怕的事情。嘿，今晚就不做什么了，写点文字，听会歌，看会书，要是再有点时间，就在看会动漫得了，上一周一集 20 多分钟动漫，看了我整整一周也都没看完，想来也真是离谱。

七月你好，七月再见！

题外话

要说今年什么最火的话，想必 AI 当然不让了，在编程领域里，在 AI 加持下开发的效率也似乎突飞猛进了起来。AI 是提效的，这点不可否认的。最初我一直推崇古法编程的，慢慢地我开始尝试了 AI 编程，配合 GitHub Copilot ，一下子看起来代码行数上了去，任务交付也提速了不少。早期我使用 Github Copilot 十分克制，都是 ASK 模式，我会仔细 Review 来自 AI 的每行代码，发现不对劲的地方还会进行多轮对话，后来试着用 Agent 模式，简单地几句话， AI 就可以把任务完成的差不多，但是带来的问题是修改的文件 ASK 模式下的从一个接着一个，一下子变成了几个文件上百甚至上千行代码，我开始放弃了 Review，一路 Accept，沉溺于提效的幻觉中。结果是人需要为自己的懒惰付出代价的，AI 看似让写代码快了很多，但是不知不觉会引入很多不易察觉的问题，一个一个堆积起来细小的问题总就会在某天爆发出来，那个时候多半也为时晚矣～

野草
