由foreach中使用引用导致的问题

By LeetaoGoooo on July 16, 2023

问题

看下面一段代码,如果你能够很清楚的知道结果是什么,就不需要阅读本篇文章了.

<?php
$array = [
    'name' => 'php',
    'age' => 123,
];
 
foreach ($array as $key => &$value) {
    echo "key=$key, value=$value" . "</br>";
}

echo "<br>"

foreach ($array as $key => $value) {
    echo "key=$key, value=$value" . "</br>";
}
?>

正确输出如下:

key=name, value=php
key=age, value=123

key=name, value=php
key=age, value=php

解答

在解释产生上述结果原因之前我们先看下面一段代码

<?php
$array = [
    'name' => 'php',
    'age' => 123,
];
 
foreach ($array as $key => $value) {
    echo "key=$key, value=$value" . "</br>";
}

echo "<br>"

foreach ($array as $key => $value) {
    echo "key=$key, value=$value" . "</br>";
}

?>
